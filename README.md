# perrr

Native Vitest environment with real DOM, real CSS cascade, real layout, and a virtual frame scheduler.

## Status

v0.1 in progress. See [`specs/overview/00-tdd.md`](specs/overview/00-tdd.md).

## Development

Requires:

- Rust 1.95 (pinned in `rust-toolchain.toml`)
- Node 20+
- pnpm 9+

```bash
pnpm install
pnpm build
pnpm test
```
